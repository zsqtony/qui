最后简单的写了下判断，这样一个简单且比较完善的 Tag 标签就写好了，可以直接复制粘贴到自己项目里，直接成为你的组件
当然了还有优化的空间，你可以继续优化改动他

API​

Props​

属性说明类型默认值可选值
label文字内容String--
shape设置圆角String3px根据实际开发自行设置值，如果是用于开发小程序可使用：2 倍的 rpx 为单位
如：6rpx
plain镂空透明背景Stringdarkdark/plain

最终代码​

export default function index({
  label,
  type = "dark",
  shape = "3px",
  bgColor = "#1677ff",
}) {
  return (
    <span
      style={{
        padding: "3px 6px",
        fontSize: "12px",
        backgroundColor: type == "dark" ? bgColor : "",
        color: type == "dark" ? "#fff" : bgColor,
        borderRadius: shape,
        border: `1px solid ${bgColor}`,
      }}
    >
      {label}
    </span>
  );
}
