# Tag 标签

import XSTag from "@site/src/components/XSTag";

自行创建好 React，tsx 的文件

#### 引入方式

在需要用到的页面或组件里，引入即可

```jsx
/* 根据实际开发路径为准 */
import XSTag from "@/components/XSTag";
```

最后简单的写了下判断，这样一个简单且比较完善的 Tag 标签就写好了，可以直接复制粘贴到自己项目里，直接成为你的组件<br />
当然了还有优化的空间，你可以继续优化改动他

#### API

#### Props

| 属性  | 说明         | 类型   | 默认值 | 可选值                                                                              |
| ----- | ------------ | ------ | ------ | ----------------------------------------------------------------------------------- |
| label | 文字内容     | String | -      | -                                                                                   |
| shape | 设置圆角     | String | 3px    | 根据实际开发自行设置值，如果是用于开发小程序可使用：2 倍的 rpx 为单位<br />如：6rpx |
| plain | 镂空透明背景 | String | dark   | dark/plain                                                                          |

#### 最终代码

```jsx
export default function index({
  label,
  type = "dark",
  shape = "3px",
  bgColor = "#1677ff",
}) {
  return (
    <span
      style={{
        padding: "3px 6px",
        fontSize: "12px",
        backgroundColor: type == "dark" ? bgColor : "",
        color: type == "dark" ? "#fff" : bgColor,
        borderRadius: shape,
        border: `1px solid ${bgColor}`,
      }}
    >
      {label}
    </span>
  );
}
```
