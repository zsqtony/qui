#### API

##### Props

| 属性    | 说明     | 类型   | 默认值  | 可选值                                                                              |
| ------- | -------- | ------ | ------- | ----------------------------------------------------------------------------------- |
| label   | 文字内容 | String | -       | -                                                                                   |
| shape   | 设置圆角 | String | 3px     | 根据实际开发自行设置值，如果是用于开发小程序可使用：2 倍的 rpx 为单位<br />如：6rpx |
| width   | 设置宽度 | String | 80px    | 根据实际情况设置                                                                    |
| height  | 设置高度 | String | 32px    | 根据实际情况设置                                                                    |
| bgColor | 背景色   | String | #1677ff | 根据实际情况设置                                                                    |

#### Event 事件

| 事件名  | 说明     |
| ------- | -------- |
| onClick | 单击事件 |

#### 最终代码

直接复制，即可使用

```jsx
import React from "react";
import "../../css/button.css";

export default function index({
  label,
  onClick,
  width = "80px",
  height = "32px",
  shape = "3px",
  bgColor = "#1677ff",
}) {
  return (
    <button
      className="qui-btn"
      onClick={() => onClick?.()}
      style={{
        borderRadius: shape,
        backgroundColor: bgColor,
        width: width,
        height: height,
        lineHeight: height,
      }}
    >
      {label}
    </button>
  );
}
```
